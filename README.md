# Cardiac Arrest Prediction System

This project focuses on predicting the risk of cardiac arrest using machine learning techniques.

## Overview
The system analyzes patient vital signs using an LSTM model to predict cardiac arrest 10–30 minutes in advance.

## Features
- Time-series prediction using LSTM
- Data preprocessing and feature engineering
- Model explainability using SHAP
- Early warning system for high-risk patients

## Technologies Used
- Python
- TensorFlow / PyTorch
- NumPy, Pandas
- Scikit-learn

## Results
- Accuracy: 98.3%
- ROC AUC: 0.999
